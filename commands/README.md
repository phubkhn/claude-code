# Commands Directory

Thư mục này dành cho danh sách command chuẩn khi dùng Claude Code cho development.

## Nhóm command gợi ý

- Setup: cài môi trường, dependencies
- Validate: lint, test, type-check
- Release: changelog, tag, publish

## Mẫu tài liệu command

Mỗi command nên ghi:
- tên command
- mục đích
- input flags
- ví dụ chạy
- expected output
