## Suggested Layout

```text
skills/
└── skill-name/
    ├── SKILL.md
    ├── examples.md
    └── templates/
```

## What A `SKILL.md` Should Include

- Skill goal
- Expected input
- Expected output
- Step-by-step workflow
- Quality checklist

## Rules

- Each skill should solve one clear use case.
- Keep examples short and practical.
