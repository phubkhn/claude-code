# Skills Directory

Thư mục này chứa các skill cho Claude Code.

## Định dạng gợi ý

```text
skills/
└── skill-name/
    ├── SKILL.md
    ├── examples.md
    └── templates/
```

## Nội dung nên có trong `SKILL.md`

- Mục tiêu skill
- Input kỳ vọng
- Output kỳ vọng
- Workflow từng bước
- Checklist chất lượng

## Quy tắc

- Mỗi skill chỉ giải quyết một use case rõ ràng.
- Ví dụ phải ngắn và chạy được trong thực tế.
